Live API Configuration

1. Production Best Practices

  • Process Management: Use PM2 for process monitoring and automatic restarts.
  • Reverse Proxy: Use Nginx to handle SSL and serve the API on standard ports (80/443).
  • Database Security: Ensure MongoDB is not accessible from the public web; use IP whitelisting.

2. PM2 Configuration

  • npm install pm2 -g
  • pm2 start server.js --name pixel-ai-api
  • pm2 save

3. Environment Hardening

In a live environment, ensure PUBLIC_URL is set to your actual domain.

  • PUBLIC_URL=https://api.yourdomain.com
  • ALLOWED_ORIGINS=https://app.yourdomain.com
