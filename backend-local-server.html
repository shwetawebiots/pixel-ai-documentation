Local Development Server

The Pixel AI backend is a robust Node.js (Express) application. It handles authentication, AI model orchestration, task queuing with BullMQ, and social media webhooks.

Step 1: Installation

Navigate to the pixel-ai folder and install dependencies.

  • cd pixel-ai
  • npm install

Step 2: Environment Configuration

Create a .env file in the backend root based on the following template.

  • NODE_ENV=development
  • PORT=3000
  • MONGODB_URI=mongodb://localhost:27017/pixel-ai
  • JWT_SECRET=your_jwt_secret
  • REDIS_URL=redis://localhost:6379
  • FRONTEND_URL=http://localhost:3001
  • # AI API Keys
  • GEMINI_API_KEY=your_gemini_key
  • GROK_API_KEY=your_groq_key

Step 3: Database & Queue Setup

  • MongoDB: Start your local MongoDB instance.
  • Redis: Ensure Redis is running for BullMQ background tasks.

Step 4: Launching the API

  • npm run dev

The backend server will run on http://localhost:3000.
