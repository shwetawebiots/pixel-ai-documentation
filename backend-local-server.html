Local Development Server

The Pixel AI Backend is a high-performance Node.js engine built with Express. It orchestrates AI model workflows, manages MongoDB persistence, and handles background tasks via BullMQ.

01

Initial Setup & Dependencies

Begin by navigating to the core backend repository and installing all necessary server-side packages.

# Navigate to backend
cd pixel-ai

# Install packages
npm install
02

System Configuration (.env)

Create a .env file in the pixel-ai root directory and configure the following core variables:

Key Purpose Default Value
PORT The port the API will listen on. 3000
MONGODB_URI Local or Atlas MongoDB string. mongodb://localhost:27017/pixel-ai
REDIS_URL Required for background AI tasks. redis://localhost:6379
GEMINI_API_KEY Your Google AI Studio key. Get Key
GROK_API_KEY Your GROQ Cloud platform key. Get Key

Security Note: Never commit your .env file to version control. Keep your API keys and JWT secrets private at all times.

03

Execute Server

Start the server in development mode with automatic hot-reloading.

npm run dev

The API will be active at: http://localhost:3000
