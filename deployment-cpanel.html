Deploying API to CPanel

1. Setup Node.js App

In your CPanel, search for Setup Node.js App. Click Create Application.

2. Application Settings

  • Node.js version: Choose the latest LTS (18+).
  • Application Root: Specify your project folder (e.g., api_pixelai).
  • Application URL: Select your subdomain.
  • Startup file: Set to server.js.

3. Upload & Install

Upload your pixel-ai backend files (excluding node_modules) via File Manager. Back in the Node.js App interface, click Run NPM Install.

4. Environment Variables

Scroll down to Environment variables and add all keys from your .env file.
