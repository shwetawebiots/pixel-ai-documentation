CPanel Application Hosting

CPanel is a popular choice for traditional web hosting. Follow these steps to deploy the Pixel AI Backend using the Node.js Selector.

1
Prepare Node.js App

Login to your CPanel dashboard, navigate to the Software section, and select Setup Node.js App. Click Create Application.

2
Application Settings
  • Node.js Version: Select 18.x or the latest LTS.
  • Application Mode: Set to Production.
  • Application Root: Path where you uploaded files (e.g., api_pixelai).
  • Startup File: Set this to server.js.
3
Deployment & Install

Upload your pixel-ai backend files (excluding node_modules) via File Manager. Once uploaded, return to the Node.js interface and click Run NPM Install.

4
Environment Variables

In the Node.js App interface, scroll down to Environment Variables and manually add all keys from your .env file (e.g., MONGODB_URI, GEMINI_API_KEY, etc.).
