Deployment

1. Frontend Deployment (Vercel)

Deploying the Next.js frontend is easiest on Vercel:

  • Push your code to a GitHub repository.
  • Import the repository into Vercel.
  • Add the necessary Environment Variables in the Vercel dashboard.
  • Click Deploy.

2. Backend Deployment (VPS/CPanel)

For the Node.js backend, you can use a VPS with PM2:

# Install PM2 globally
npm install pm2 -g

# Start the application
pm2 start server.js --name pixel-ai-backend

# Ensure it starts on reboot
pm2 save
pm2 startup

Ensure your firewall allows traffic on the configured port (default 3000).
