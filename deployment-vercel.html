Deploying your Next.js frontend

1. Push to GitHub

First, ensure your pixel-ai-next code is pushed to a GitHub, GitLab, or Bitbucket repository.

2. Connect Vercel

Log in to Vercel and click New Project. Import your repository.

3. Configure Project

In the Environment Variables section, add all keys from your .env.local:

  • NEXT_PUBLIC_API_BASE_URL: Your live API URL (e.g., https://api.pixel-ai.com/api).
  • NEXT_PUBLIC_SOCKET_URL: Your live socket URL (e.g., https://api.pixel-ai.com).

4. Deploy

Click Deploy. Vercel will automatically build and serve your application with global CDN benefits.
