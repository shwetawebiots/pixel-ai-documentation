Command Center

The Pixel AI Admin Dashboard is the central hub for managing the entire platform. It provides high-level control over users, financial tiering, and system-wide API integrations.

User & Role Management

Full control over the user lifecycle. Create, block, or delete users and assign specialized roles with granular permissions.

  • RBAC: Define custom roles (Admin, Moderator, Editor).
  • Activity Logs: Monitor login and AI usage history.
Subscription & Plans

Manage the platform's revenue streams. Define what each tier can access.

  • Token Budget: Set weekly/monthly AI word limits per plan.
  • Feature Gating: Enable/Disable specific tools based on user tier.
