Prerequisites for Pixel AI

System Requirements

Before installing Pixel AI, ensure your server or local environment meets the following baseline requirements:

  • Node.js: Version 18.x or higher (LTS recommended).
  • MongoDB: Version 6.0 or higher.
  • Redis: Required for task queuing (BullMQ) and performance caching.
  • NPM / Yarn: Version 8+ or 1.22+.

Mandatory API Keys

To enable AI features, you must obtain API keys from the following providers:

Provider Feature Link
Google Cloud Gemini Pro AI (Text/Vision) Get Key
GROQ Cloud High-speed Llama3/Mixtral Get Key
OpenAI GPT-4 / DALL-E (Optional) Get Key

Third-Party Integrations

For payments and social media features, the following are required:

  • Stripe / Razorpay / Paypal: Developer accounts for subscription management.
  • Facebook Developer: App with Instagram Graph API permissions for social automation.
  • SMTP Server: Gmail (with App Password) or Mailtrap for system emails.
