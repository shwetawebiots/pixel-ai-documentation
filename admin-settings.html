System Settings

The System Settings module provides high-level control over the core AI engine, payment gateways, and platform branding.

Step 1: AI Engine Configuration

Enter your API keys for OpenAI, Google Gemini, and Anthropic. These keys power the entire platform's AI capabilities, from chat completions to image generation.

[IMAGE: AI API Keys Configuration Interface with secure input fields]

Step 2: Payment Gateway Setup

Configure Stripe Webhooks and PayPal Client IDs to enable automated billing, subscription management, and secure transactions.

[IMAGE: Finance & Payment Integration Settings for Stripe and PayPal]

Step 3: Social Auth & Branding

Set up OAuth for Google and Facebook login, and upload your custom Logo and Favicon to ensure consistent brand identity across the platform.

[IMAGE: Brand Identity settings with Logo upload and Social Login Config]

Step 4: Regional & Misc Settings

Set the platform's default language, timezone, and currency symbols to cater to your specific target audience and market.

[IMAGE: Localization preferences for Language, Timezone, and Currency]
