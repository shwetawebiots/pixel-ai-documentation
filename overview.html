Pixel AI Installation

Architecture Overview

1. Project Structure:

Pixel AI is divided into two primary repositories: pixel-ai (Backend API) and pixel-ai-next (Frontend). Both must be configured correctly for the platform to function.

2. Installation Order:

We recommend setting up the Backend first to ensure the database and Redis services are active, followed by the Frontend configuration.

Pixel AI Bundle Composition:

  • 1. pixel-ai (Node.js/Express Backend)
  • 2. pixel-ai-next (Next.js 16 Frontend)
  • 3. pixel-ai-documentation (Static HTML Docs)

Follow the subsequent guides for detailed environment-specific installation steps.
