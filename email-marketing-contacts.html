Contact & Segment Management

Audience Control

Effective email marketing starts with a well-organized audience. This module allows you to manage your subscribers, group them into static lists, or create dynamic segments for highly targeted messaging. Follow the steps below to master audience management.

1. Individual Contact Management
Step 1: Adding New Contacts

Click the "Create Contacts" button to launch the contact creation modal. Here, you must provide the subscriber's email address. You can also add their name and descriptive tags for better organization and segmenting later.

[IMAGE: Create Contact Modal with Email, Name, and Tags fields]

Step 2: Editing Contact Attributes

To update an existing contact's details or change their tags, click the Pencil Icon in the Actions column. This opens the edit modal where you can modify any piece of information stored in the contact record.

[IMAGE: Edit Contact Modal showing existing data being updated]

Step 3: Secure Data Removal

If a subscriber needs to be removed from your database, use the Trash Icon. A confirmation prompt will appear—once confirmed, the contact and all their associated data will be permanently deleted from the system.

[IMAGE: Contact List View focusing on the Delete action and confirmation prompt]

2. Contact List Management (Static)
Step 1: Defining the List

Use the "Create Contact List" button to define a new static grouping. Provide a clear Name (e.g., "Monthly Newsletter") and an optional Description to help team members understand the list's purpose.

[IMAGE: Create Contact List Modal - Name and Description fields]

Step 2: Assigning Members

Within the creation modal, use the "Select Contacts" section to search and pick subscribers from your central repository to add to this specific list. You can add as many as needed.

[IMAGE: Contact Selection UI within the List Creation modal]

Step 3: List Actions & Maintenance

From the main Contact List view, you can use the User Group Icon to quickly see all members of a list. The Edit and Delete icons allow for quick metadata updates or complete list removal.

[IMAGE: Contact List Toolbar showing Edit, View Members, and Delete icons]

3. Dynamic Segment Management
Step 1: Segment Configuration

Start by clicking "Create Segment" and entering a unique Name. Segments are dynamic, meaning they automatically include any contacts that match your defined rules in real-time.

[IMAGE: Create Segment Modal - Initial Name field]

Step 2: Building Rule-Based Criteria

Add conditions using the Criteria builder. Select a Field (Email, Name, Tags), an Operator (contains, equals, starts with), and a Value. For example, to target all Gmail users, set: Email | contains | gmail.com.

[IMAGE: Criteria Builder Interface showing dynamic rule selection]

Step 3: Lifecycle & Syncing

Once created, the segment constantly scans your database. If a new contact matches the criteria, they are added automatically. You can refine rules via the Edit Icon or remove the segment using the Delete Icon.

[IMAGE: Segment List View with sync status and action buttons]
